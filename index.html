오피스가드 설명

OfficeGuard는 업무망에서 허용할 사이트와 차단할 사이트를 분리하고, 사용자와 조직별 정책에 맞춰 인터넷 사용을 관리하는 웹 보안 솔루션입니다.

성능 및 안정성

1. ETAP 기반 고성능 SSL 복호화 엔진

2. DPI 기반 트래픽 분석

3. 40Gbps 환경 성능 지원

4. 듀얼 링크 및 비대칭 네트워크 지원

SSL 복호화 및 가시성

1. Full Transparent SSL/TLS 복호화

2. SSL 가시성 기능 제공

3. 다양한 3rd Party 연동

4. 장비 맞춤 복호화 지원

운영 편리성

1. 직관적인 정책 설정 UI

2. 사용자 편의성 강화

3. 암·복호화 실패 대응

4. 다양한 바이패스 기능 지원

오피스가드 종류

오피스가드는 기본 차단, SSL 가시성, 통합형 보안으로 나뉘며, 각각의 제품은 서로 다른 운영 환경과 보안 요구 수준에 맞춰 설계되어 있습니다.

플랜티넷 소개

플랜티넷은 네트워크 전문 솔루션 기업으로서 국내·외에서 검증받은 기술력을 기반으로 안전한 인터넷 세상을 만들어가겠습니다.
