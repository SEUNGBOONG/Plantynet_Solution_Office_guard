# OfficeGuard Solution Website

플랜티넷 OfficeGuard / OfficeGuard SV / OfficeGuard Plus 제품군을 소개하는 반응형 홍보 사이트입니다.

## 실행

이 사이트는 정적 HTML/CSS/JavaScript로 구성되어 있어 `index.html`을 브라우저에서 바로 열 수 있습니다.

개발 서버가 필요한 경우 Node.js가 설치된 환경에서 다음 명령을 사용할 수 있습니다.

```bash
npm start
```

## PDF Export

powershell -ExecutionPolicy Bypass -File .\tools\export-pdf.ps1
